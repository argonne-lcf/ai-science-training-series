# Homework

Implement your own multi-agent with your own tools and demonstrate with ALCF endpoints.

## Submission Requirements

Submit two or optionally three files:
- `<your_name>_tools.py`
- `<your_name>_multi_agent.py`
- Optionally: `requirements.txt`

## Requirements

- Make sure you make use of ALCF endpoints
- Your code must run with `python <your_name>_multi_agent.py`
- Any additional dependency required should be listed in `requirements.txt` and must be installable by pip
